# 滑块和点选图片坐标计算(rust)

## 构建方法

```sh
maturin build
```

运行上述命令，即可在`target/wheels`文件夹下面得到对应的Python whl包。

如果不想自己构建，可以采用我构建好的项目, 这里对于Python版本有要求

- mac(intel): python3.8
- win: python3.10
- ubuntu: python3.8

其他平台目前我没有测试和构建过, m1的mac我这边没有, 有用这个的自行构建吧。
